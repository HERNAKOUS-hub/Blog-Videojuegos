# Blog de Videojuegos 🎮

Proyecto de Front-end desarrollado para el ciclo de 2º DAW. Consiste en una *landing page* interactiva y moderna para un blog de noticias, reseñas y tendencias del mundo de los videojuegos.

## 🚀 Características Principales

* **Estructura Semántica:** Uso de etiquetas HTML5 (`<header>`, `<nav>`, `<section>`, `<aside>`) para mejorar la accesibilidad y el SEO.
* **Diseño Fluido (Flexbox):** Maquetación de tarjetas, galerías e interfaz utilizando Flexbox para una alineación perfecta de los elementos.
* **Interfaz Interactiva (UI):** * Efectos de transición (*hover*) avanzados en las tarjetas de juegos destacados utilizando `clip-path` y variaciones geométricas.
  * Menú de navegación pegajoso (*sticky*) en la sección del blog para facilitar la lectura.
  * Botón fijo (*Back to top*) con comportamiento de desplazamiento suave (*smooth scrolling*).
* **Gestión de Estilos:** Uso de Variables CSS (`:root`) para mantener una paleta cromática coherente y facilitar futuros temas (dark/light mode).

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura y semántica.
* **CSS3:** Estilos, variables, animaciones, transiciones y gradientes.
* **FontAwesome:** Iconografía vectorial para redes sociales y elementos de la interfaz.

## 📂 Estructura del Proyecto

- `index.html`: Documento principal de la vista.
- `BlogVideojuegos.css`: Hoja de estilos principal.
- `/img`: Directorio de recursos gráficos y fotografías.
- `/fontawesome-free...`: Directorio de dependencias para iconografía.
